
Website Design Prompt: Budget Management Website

Objective:
Create a responsive and user-friendly website for managing personal or family budgets, tailored to the Nigerian economy.

Color Scheme:
- Primary: Green (#008000) for highlights, buttons, and headers.
- Secondary: White (#FFFFFF) for backgrounds and text areas.
- Accent: Light gray for borders, Red (#FF0000) for expenses.

Typography:
- Font: Poppins or Roboto.
- Title font size: 24–32px.
- Text font size: 14–18px.

Wireframe Layout:

1. Homepage (User Selection):
   - Header Section:
     - Title: "Welcome to Budget Manager"
     - Subtitle: "Track and manage your finances effortlessly."
   - Main Section:
     - Two large cards/buttons for user type selection:
       - Individual: Icon of a wallet, text: "For personal finance management."
       - Parent: Icon of a family, text: "Manage your family’s expenses."
     - Hover animations for buttons (e.g., scale up slightly).
   - Footer Section: Links to "Features", "About", "Contact".

2. Dashboard Screen:
   - Header Section:
     - Greeting: "Hello, [User Name]"
     - Subtitle: "Here’s a summary of your finances."
   - Main Section:
     - Summary Cards:
       - Total Income: Green card, bold white text (e.g., "₦50,000").
       - Total Expenses: Red or gray card, bold white text.
       - Balance: Highlighted in green or red based on value.
     - Expense Distribution Chart: Pie chart showing expense categories (Food, Transport, etc.).
   - Quick Links: Buttons for "Add Income" (Green) and "Add Expense" (Red).

3. Add Income/Expense Form:
   - Title: "Add Income or Expense."
   - Fields:
     - Description: e.g., "Salary, School Fees."
     - Amount: Numeric input field.
     - Dropdown: Categories like Food, Transport, etc.
   - Buttons: "Add Income" (Green), "Add Expense" (Red).

4. Parent Dashboard (Child Tracking):
   - Header Section:
     - Title: "Child Expense Tracker."
   - Main Section:
     - Add Child Button: Prominent button labeled "Add a Child."
     - Child List: Each child displayed as a card showing:
       - Name (e.g., "Child 1").
       - Expense categories with amounts (e.g., "School Fees: ₦20,000").

5. Savings Goals (Optional):
   - Header Section:
     - Title: "Savings Goals."
   - Main Section:
     - List of Goals: Progress bar (e.g., "₦5,000 of ₦20,000 saved").
     - Edit/Delete buttons for each goal.

Responsive Design:
- Desktop: 12-column grid with 80% width content, centered.
- Mobile: Single-column layout with full-width buttons and inputs.

Tools to Use:
- Design: Figma, Adobe XD, or Sketch.
- Plugins: "Feather Icons," "Material Design Icons," "Chart" for graphs, and "Coolors" for color palette.

Interactions:
- Use Figma’s prototype tool to link buttons and pages.
- Add hover effects to buttons and cards.
